InDhanPay App

For both IOS & Android

It is based on the core code of Dart,its a flutter developed App for the future payments. InDhanPay is using an intermediary back end to verify the transaction log & ensure the security of each transaction. The intermediary back end is based on Node.js code and then the data is shared to the back end of MySQL as the main database.
